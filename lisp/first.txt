LISP Functions

   (first coll)
     __________________________________________________________________

   Returns returns the first element of coll.
     __________________________________________________________________

   This function takes a list or vector as its argument and returns the first element. If the list or vector is empty or is nil then nil is returned.

EXAMPLES

   Code             Returns
   (first '(1 2 3)) 1
   (first [1 2 3])  1
