LISP Functions

   (count coll)
     __________________________________________________________________

   Returns the total number of items present in that collection.
     __________________________________________________________________

   The first parameter as a list or vector and return the number of elements that it contains. The resulting number is an integer.

EXAMPLES
   Code             Returns
   (count '(1 2 3)) 3
   (count [1 2 3])  3
