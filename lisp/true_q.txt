LISP Functions

   (true? x)
     __________________________________________________________________

   Returns true if x is the value true, false otherwise.
     __________________________________________________________________

   This function takes a single argument and returns true if the argument is a true value.

EXAMPLES
   Code          Returns
   (true? true)  true
   (true? 1)     false
