   Present the OK, button.

   DCL Error Handler Tile
     __________________________________________________________________

   This error handler tile features a button labeled OK. It is associated with the pre-defined key "accept." This tile is particularly beneficial for presenting a message or alert that does not provide any alternative options. The user is required to click the button to acknowledge the message or alert and to dismiss the dialog box.

   To display only the OK button in a dialog box (for EXAMPLE, in a message or alert box), use the [link]ok_only.

NOTE
   For most dialog boxes, you'll want to use the [link]ok_cancel tile instead of this tile.

Valid Attributes
   none

EXAMPLE
   ok_only;
